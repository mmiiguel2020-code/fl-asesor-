# CHANGELOG

## v1.0.0 (2026-07-09)

### Features ✨
- ✅ Conversión a aplicación de escritorio independiente con Electron
- ✅ Captura nativa de pantalla con soporte multi-display
- ✅ Análisis inteligente con IA Gemini 3.5 Flash
- ✅ Base de datos SQLite para historial persistente
- ✅ Chat conversacional de asesoramiento en FL Studio
- ✅ Notificaciones nativas del escritorio
- ✅ Atajos de teclado optimizados
- ✅ Tema claro/oscuro adaptativo
- ✅ Interfaz responsive y moderna con Tailwind CSS

- Mejor gestión de memoria con caché
- DevTools integrado en desarrollo
